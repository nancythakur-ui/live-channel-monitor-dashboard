# 📡 Live Channel Monitor Dashboard

A web-based dashboard to monitor Live TV / OTT channels and detect working, freeze, or blank streams.

## 🚀 Features

- Paste Live Channel URL and check status
- Upload CSV file to monitor multiple channels
- Detect blank screen streams
- Detect paused / freeze channels
- Smooth playback detection
- Daily / Weekly / Monthly reports
- Working vs Not Working filters
- Download CSV reports
- Trending JioHotstar scraper
- Real-time stream monitoring
